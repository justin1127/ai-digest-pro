- 🌐 访问地址：https://justin1127.github.io/ai-digest-pro/

- 🔐 访问控制：专属密码（自动生成）

## 工作流程

```
用户看到广告/内容 → 加你微信 → 你发送收款码 → 用户付款

你在本地生成密码 ─→ 发送给用户 ─→ 用户输入密码解锁日报
```

## 快速开始

### 1. 生成用户密码

```bash
cd /Users/maxjustin/.openclaw/workspace/ai-digest-pro
python3 generate_password.py
```

按提示输入：
- 用户姓名
- 微信名  
- 付费类型（月付/季付/年付/永久）

系统会：
- 生成专属密码（如：`AI26A3B9C2D1E`）
- 保存到 `subscribers.ini`
- 输出发送给用户的微信话术

### 2. 查看所有订阅用户

```bash
python3 manage_passwords.py
```

显示：
- 所有用户密码
- 有效期状态
- 付费类型

### 3. 生成并发布日报

```bash
python3 generate.py
```

这个命令会：
- 读取 `subscribers.ini` 中的有效密码
- 生成带密码保护的HTML页面
- 更新 `index.html` 跳转

然后手动推送到GitHub：
```bash
git add .
git commit -m "AI决策情报 2026.03.17"
git push
```

## 注意事项

1. **密码安全**：`subscribers.ini` 包含所有用户密码，不要上传到公开仓库
2. **定期备份**：建议定期备份 `subscribers.ini` 文件
3. **密码过期**：系统会自动检查有效期，过期密码无法访问
4. **内容更新**：每天需要更新 `generate.py` 中的 `TODAY_NEWS` 变量
