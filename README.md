### 免费版
- 每天3条AI日报摘要
- 最多10个频道
- 7天历史记录
- 基础推送

### 付费版（¥29/月）
- 无限量AI日报
- 全部50个频道
- 完整历史记录
- 深度分析报告
- 关键词定制
- 个性化推送时间

## 技术栈

- **前端**：Next.js 14 + React + Tailwind CSS
- **后端**：Next.js API Routes
- **数据库**：PostgreSQL + Prisma ORM
- **支付**：Stripe + 微信支付
- **推送**：飞书Bot + 微信公众号
- **部署**：Vercel / 阿里云

## 快速开始

### 1. 安装依赖
```bash
cd ai-digest-pro
npm install
```

### 2. 配置环境变量
```bash
cp .env.example .env.local
# 编辑 .env.local 填写配置
```

### 3. 初始化数据库
```bash
npx prisma db push
npx prisma generate
```

### 4. 启动开发服务器
```bash
npm run dev
```
